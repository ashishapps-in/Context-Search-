## Install Context Search+ in 3 Easy Steps

### Step 1: Download
Download all files from the `context-search-plus` folder to your computer.

### Step 2: Open Chrome Extensions
1. Open Google Chrome
2. Type `chrome://extensions/` in the address bar
3. Press Enter

### Step 3: Load the Extension
1. Toggle **"Developer mode"** ON (top-right corner)

## Troubleshooting

**Bubble not appearing?**
- Refresh the webpage (F5)
- Make sure you're highlighting more than 1 character
- Check if you're on a restricted page (chrome:// pages don't allow extensions)

**Context menu not showing?**
- Right-click the extension icon â†’ Check permissions

**Translation not working?**
